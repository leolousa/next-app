# next-app

Aplicação de testes do Framework React Next.js


## Instalação do Next.js
```bash
npm install next react react-dom
# ou
npx create-next-app
```

## Usar Typescript (.jsx)
```bash
npm install --save-dev typescript @types/react @types/node
```

## Usar Axios (chamadas para API)
```bash
npm install axios
```
